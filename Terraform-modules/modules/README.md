# Modules

This folder contains **reusable Terraform modules** that help reduce code duplication and improve maintainability.

## 📂 Modules Available

```bash
modules/
├── vpc/              # Networking (VPC, subnets, routing)
├── ec2/              # EC2 compute instances
├── sg/               # Network security rules
├── eks/              # Kubernetes (EKS) cluster
├── alb/              # Application Load Balancer
├── rds/              # Relational Database Service
├── s3/               # Object storage (S3 buckets)
└── asg/              # Auto Scaling Groups

```
